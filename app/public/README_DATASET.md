# Opis skupa podataka: HNL Igrači

Ovaj skup podataka sadrži informacije o igračima Hrvatske nogometne lige (HNL) za sezonu 2023/2024.

## Metapodaci

- **Autor:** Luka Kordić
- **Verzija:** 2.0
- **Jezik:** Hrvatski
- **Licenca:** Creative Commons Imenovanje-Nekomercijalno-Dijeli pod istim uvjetima 4.0 međunarodna
- **Atributi:**
    - `klub`: Naziv kluba
    - `godina_osnutka`: Godina osnutka kluba
    - `stadion`: Naziv stadiona
    - `igraci`: Lista igrača u klubu
        - `ime`: Ime igrača
        - `prezime`: Prezime igrača
        - `pozicija`: Pozicija igrača
        - `broj_dresa`: Broj dresa igrača
        - `nacionalnost`: Nacionalnost igrača
        - `vrijednost_eur`: Tržišna vrijednost igrača u EUR
